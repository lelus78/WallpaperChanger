## üîß Troubleshooting

### "Please configure Gemini API key first"
**Solution**: Enter and save your API key in the AI Assistant tab

### "No wallpaper selected"
**Solution**: Click the file picker button to select a wallpaper first

### "AI search failed"
**Possible causes**:
- No internet connection
- Invalid API key
- API rate limit reached (free tier: ~60 requests/minute)

**Solution**: Check your connection, verify API key, or wait a minute

### Slow AI responses
**Possible causes**:
- Slow internet connection
- Large cache requiring more analysis

**Solution**: Normal response time is <2 seconds. If consistently slower, check your network

## üìä Understanding AI Scores

### Similarity Score (0-100%)
- **90-100%**: Nearly identical style/composition
- **80-89%**: Very similar artistic approach
- **70-79%**: Similar themes with some variation
- **60-69%**: Related styles
- **Below 60%**: Different but potentially interesting

### Prediction Confidence (0-100%)
- **80-100%**: High confidence - very likely you'll love it
- **60-79%**: Good match based on patterns
- **40-59%**: Reasonable suggestion
- **Below 40%**: Experimental recommendation
