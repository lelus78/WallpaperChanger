## Quick Start (For Beginners)

### Step 1: Install Python
1. Go to https://www.python.org/downloads/
2. Download Python (latest version)
3. **IMPORTANT:** During installation, check "Add Python to PATH"
4. Click "Install Now"

### Step 2: Install Wallpaper Changer
1. Double-click **`INSTALL.bat`** in this folder
2. Wait for installation to complete (may take 1-2 minutes)
3. Choose if you want it to start automatically with Windows

### Step 3: Start Using
- **Option 1:** Double-click "Wallpaper Changer" icon on your desktop
- **Option 2:** Press Windows key, type "Wallpaper Changer", press Enter

## Common Questions

### "How do I get new wallpapers?"
- Press **Ctrl+Alt+W** anytime
- Or click **Change** in the Home tab
- Or wait for automatic change (if configured)

### "How do I view a wallpaper in full size?"
- Go to **Wallpapers** tab
- Click on any thumbnail
- Press **Escape** to close

### "How do I favorite a wallpaper?"
- In **Wallpapers** tab, click the ‚ô° button
- Or open fullscreen view and click **Add to Favorites**

### "How do I filter by tag?"
- Go to **Wallpapers** tab
- Use the **Tag:** dropdown at the top
- Select a tag to see only wallpapers with that tag

### "Where are my wallpapers stored?"
- `C:\Users\YourName\WallpaperChangerCache`

### "How do I uninstall?"
- Double-click **`UNINSTALL.bat`** in this folder
- Or use the shortcut in Start Menu

## Need Help?

### The program won't start
1. Make sure Python is installed
2. Try running `INSTALL.bat` again
3. Check if antivirus is blocking it

### Wallpapers not changing
1. Open the GUI
2. Check **Logs** tab for errors
3. Make sure you have internet connection
4. Verify API keys in **Settings** tab (if using Pexels)

### Report an Issue
- Visit: https://github.com/anthropics/claude-code/issues
- Include error messages from **Logs** tab
